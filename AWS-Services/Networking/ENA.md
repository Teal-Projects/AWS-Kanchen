# Description 

Amazon EC2 provides enhanced networking capabilities through the Elastic Network Adapter (ENA). To use enhanced networking, you must install the required ENA module and enable ENA support.

# Main Features 
 
### General : 
* Use signle root I/O virtualization (SR-IOV) to provide high-performance networking capabilities on supported instance types. Higher packet per second. There is additional charged for using htis network. 
* Can be enabled by : 
	* Elastice network Adapter(ENA), network spedd up to 100 Gbps.
